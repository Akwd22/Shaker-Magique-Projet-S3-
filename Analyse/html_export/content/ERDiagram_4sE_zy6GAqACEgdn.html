stocker Table
Column Name Data Type PK / FK Nullable Description
enReserve bool No Indique si l'ingrédient est en stock dans la réserve

noter Table
Column Name Data Type PK / FK Nullable Description
note int2 No Note d'un cocktail sur 5 étoiles. Doit être compris entre 0 et 5 inclus

Membre Table
Column Name Data Type PK / FK Nullable Description
id int4 PK No Identifiant d'un membre
prenom varchar(32) Yes Prénom du membre
nom varchar(32) Yes Nom de famille du membre
dateN date No Date de naissance du membre
genre char(1) Yes Genre du membre. Valeurs possibles : "M", "F", NULL
idCompte int4 FK No
idHote int4 FK Yes

contenir Table
Column Name Data Type PK / FK Nullable Description
quantite varchar(16) No Quantité d'ingrédient dans le cocktail. Exemple "1/2", "3/10", "1"
unite varchar(16) Yes Unité de la quantité. Par exemple : 1 trait de citron, 1 dose d'alcool, ou NULL

Cocktail Table
Column Name Data Type PK / FK Nullable Description
id int4 PK No Identifiant d'un cocktail
intitule varchar(255) No Titre du cocktail
illustrationURL varchar(255) Yes URL de l'image d'illustration du cocktail
categorie varchar(255) No Catégorie du cocktail. Doit avoir trois valeurs possibles : "Apéritif", "Digestif", "Apéritif/Digestif"
description text Yes Description du cocktail : Verre et Déco, Mode opératoire. Peut contenir un formatage HTML. Voir modèle : https://tourdebouffe.food.blog/2020/07/05/love-cocktail/
force int2 Yes Force du cocktail

Ingredient Table
Column Name Data Type PK / FK Nullable Description
id int4 PK No Identifiant d'un ingrédient
intitule varchar(255) No Titre de l'ingrédient
degreAlcool int2 Yes Degré d'alcool de l'ingrédient. 0 <=> sans d'alcool/n'est pas de l'alcool

Compte Table
Column Name Data Type PK / FK Nullable Description
id int4 PK No Identifiant du compte
login varchar(32) No Login du compte. Ne doit pas contenir des caractères spéciaux
passHash varchar(255) No Hash du mot de passe
email varchar(255) No E-mail du compte. Doit respecter un format valide
dateCreation date Yes Date et heure de création du compte
derniereConnexion date Yes Date et heure de la dernière connexion au compte
enLigne bool No Indique si le compte est actuellement connecté
sessionId uuid Yes Identifiant unique de la session en cours. Utilisé dans le cookie de session pour reconnecter automatiquement l'utilisateur s'il a coché la case "Se souvenir de moi"
