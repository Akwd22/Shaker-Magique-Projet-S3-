Pour bien installer l'environnement de travail :

Créer un dossier où mettre le projet 

Exécuter les commandes dans git Bash à l'endroit du dossier ( cd/Desktop/.../dossierOuLeProjetSeTrouve )  :
git clone https://github.com/Taarjax/Projet-S3.git
git pull origin master

A partir de ce momment dans le dossier devrait se trouver : 
- Un dossier "Saker_Magique"
- Un fichier .gitignore
- Un fichier requirements.txt
- Ce fichier.txt

Ouvrez le CMD 
Dirigez-vous à l'endroit où le projet est. ( ex : C:\Users\Remy\Desktop\Workspace\ProjetS3 ) 

executer ses commandes dans l'odre :

pip install virtualenv 
virtualenv env ( Mettez bien "env" comme nom )
env\Scripts\activate.bat ( cela active l'environnement virtuel, il devrait y avoir (env) qui apparait 
pip install -r requirements.txt ( Cela installe toutes les librairies requises )
